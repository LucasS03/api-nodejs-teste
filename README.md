## Rodar aplicação

Primeiro é necessário criar dois arquivos na raiz do projeto:
- **.env** e **.env.examples**
- colocar em ambos os arquivos o seguinte trecho:
```
SECRET_KEY=add_alguma_chave_secreta_aqui
```

Feito isso, basta instalar as dependências com:
```
npm i
```
 
E rodar a aplicação:
```
nodemon start
```
